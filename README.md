Dictionary App

A minimalist, modern Dictionary application built with Jetpack Compose and Kotlin.
This app fetches real-time word definitions, phonetics, and usage examples using a REST API.

🚀 Features
Instant Search: Search for any English word to get detailed definitions.

Phonetics Support: View phonetic transcriptions for better pronunciation.
Categorized Definitions: Meaning grouped by parts of speech (Noun, Verb, Adjective, etc.).Usage Examples: Real-world examples for better context.
Modern UI: Clean, minimalist design following Material Design 3 guidelines with support for dark mode.

🛠 Tech Stack
Language: KotlinUI 
Framework: Jetpack 
ComposeArchitecture: MVVM (Model-View-ViewModel) + Clean ArchitectureDependency 
Injection: HiltNetworking: Retrofit & OkHttpState 
Management: Kotlin Flows & StateFlowImage 
Loading/Icons: Material Symbols & Compose Icons📸
